🎋 粽子人格测试（Who Are You, Zongzi?）

一个把端午粽子、人格测试和中国地域饮食文化结合在一起的互动小游戏。

用户通过选择不同的粽子配料，最终获得一个独属于自己的「粽子人格」。

有些配料来自真实的中国粽子文化。

有些则来自天马行空的想象。

最终包出来的，可能是一颗粽子，也可能是一个关于自己的小故事。

✨ 项目背景

端午节前，我们发现：

大家都在争论：

甜粽还是咸粽？
红豆还是蛋黄？
北方粽还是南方粽？

于是产生了一个有趣的问题：

如果人的性格是一颗粽子，它会是什么馅？

于是有了这个项目。

🧠 设计原则
1. 结果比测试更重要

用户不需要看到复杂的计算过程。

只需要得到一个有趣、愿意分享的结果。

2. 配料是真实文化入口

项目中的许多配料来自：

广东
福建
潮汕
嘉兴
台湾
广西

等不同地区的粽子文化。

人格测试只是入口。

真正有趣的是地域饮食背后的故事。

3. 奇怪配料负责惊喜

如果所有材料都很正常。

结果也会变得普通。

因此项目中加入了一些：

超现实
荒诞
情绪化

的材料。

让每个人都能包出一个现实中不存在的粽子。
